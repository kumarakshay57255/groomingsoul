Today's Updates

Four new features added to the Grooming Souls platform — shown on the website and in the admin panel.

  • 1 — Founder Achievements section
  • 2 — Founder's Book + Books page (Buy on Amazon)
  • 3 — Course cover images (thumbnails)
  • 4 — Updated social links (Instagram & LinkedIn)
Live at https://groomingsouls.com  ·  Every screenshot below is the real, working feature
1 Founder Achievements New

A new Achievements & Recognition section on the home page showing the founder's awards and milestones as cards. Your team adds them from the admin — each with an image, year, and description.

On the website — the home page “Milestones that shaped the mission” section, with a card per achievement.

In the admin — Admin → Founder Achievements: add, edit, publish/unpublish and reorder.

How it works: add an achievement, tick Published, and it appears on the home page instantly.

2 Founder's Book & Books page New

The founder's book “Unseen Minds and Hidden Souls” is now featured on the home page and on a dedicated /books page, each with a “Buy on Amazon” button. Title, cover and description were taken from the Amazon listing.

On the website — the “Published Work” band on the home page, with the real cover and a Buy on Amazon button.

In the admin — Admin → Books: cover upload, title, author, format, Amazon link, description, published toggle.

Also added: a footer link “Founder's Books” → the /books page, which lists every published book.

3 Course cover images New

Courses can now have a cover image (thumbnail) instead of only a colour. Upload a photo in the admin and it becomes the card image on the Academy/Diploma pages. No image? It falls back to the colour — nothing breaks.

On the website — the first course now shows an uploaded image; the others use the colour fallback.

In the admin — Edit course → “Cover image (recommended)” to upload, with the colour kept as a fallback.

Backward-compatible: every existing course keeps working — the image is optional.
