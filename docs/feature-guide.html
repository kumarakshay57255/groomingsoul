02 — The website

The other pages

Therapy

Browse & book therapists

Visitors see verified psychologists (photo, specialty, fee) and send a booking request — no online payment, just a conversation. Requests land in the admin as Therapy Leads.

Free Tests

5 psychometric tests

Big Five personality, cognitive aptitude, stress, emotional intelligence, and sentence completion — 134 questions total. Results are saved in the admin as Test Submissions.

Academy & Diploma

Paid courses

Course cards with cover image, price, and access period. Students pay by scanning a UPI QR and uploading the receipt; the admin approves it and unlocks the videos.

Books

Founder’s books

A dedicated page listing the founder’s book(s) with cover, description and a “Buy on Amazon” button. The newest one is also featured on the home page.

Student area

Dashboard & lessons

After a purchase is approved, students log in to watch their course videos (protected player) until the access period ends.

Trust

About, Contact, Policies

Foundation details, contact information, and Privacy / Terms / Refund pages for credibility and compliance.

04 — How it connects

You edit here → it shows there

The single most important idea: the admin and website are always in sync. A simple example with an achievement:

1 · Admin
Add “National Recognition”, upload image, tick Published
→
2 · Database
Saves it safely
→
3 · Website
A new card appears in the Achievements section

The same pattern applies everywhere:

When you……visitors immediately see
Add a team member + photoA new card in “Our Expert Team”.
Add an achievementA card in the Achievements section.
Add a book + cover + Amazon linkFeatured book on home + a card on /books.
Upload a course cover imageThe course card shows that image (else a colour).
Toggle something to DraftIt disappears from the website (kept in admin).
The “Published / Draft” switch is your on/off control. Draft = hidden from visitors but saved. Published/Live = visible on the website.

05 — Everyday tasks

How to do the common things

Log in

  1. Go to groomingsouls.com/admin
  2. Enter the admin email and password
  3. You land on the Dashboard — pick a screen from the left menu

Add a founder achievement

  1. Left menu → Founder Achievements → Add achievement
  2. Fill Title, Year, Type, Description; upload an image
  3. Keep Published on → Save. It’s now on the home page.

Add / update a book

  1. Left menu → Books → Add book
  2. Enter title, author, the Amazon link, description; upload the cover
  3. Save → it’s featured on the home page and on the /books page.

Add a course with a cover image

  1. Left menu → Courses & Lessons → create the course, then open it
  2. In “Course cover”, upload a cover image (or leave a colour)
  3. Add modules & upload lesson videos → set price → save.
